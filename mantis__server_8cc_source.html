7  * Default configuration options are given in class server_config (see mt_server_config.cc)
8  *
9  * The server can be configured in three ways, each of which can override previous values:
10  * 1. Default configuration (see class server_config in mt_server_config.cc)
11  * 2. Configuration file supplied by the user (json-formatted)
12  * 3. Command line options
13  *
14  * The server requires the following configuration values:
15  * - port (integer; must match the port used by the client)
16  * - buffer-size (integer; number of record blocks in the DMA buffer)
17  * - record-size (integer; number of samples in a block **** NOTE: this sets the number of samples in a block, even though it's called "record-size" ****)
18  *
19  * Usage:
20  * $> mantis_server config=config_file.json [further configuration]
21  *
22  * Arguments:
23  * - config (string; optional): json-formatted configuration file
24  * - further configuration: override or add new values
25  * format: name/type=value
26  * e.g.: port/i=8235
